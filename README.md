# Visão Geral de Casos de Uso: Tabuada Solares

1. Atores

Aluno: Usuário foco do sistema, que utiliza a ferramenta para aprender.

Professor: Facilitador que utiliza a ferramenta como apoio didático.

2. Casos de Uso

### UC01: Praticar Tabuada

Ator: Aluno.

Descrição: O aluno inicia o jogo e responde a sequências de multiplicações geradas aleatoriamente pelo sistema para treinar sua memória.

### UC02: Receber Feedback Visual

Ator: Aluno.

Descrição: Ao selecionar uma resposta, o aluno recebe um feedback imediato por cores (Verde para acerto, Vermelho para erro), reforçando o aprendizado sem necessidade de correção manual.

### UC03: Acompanhar Desempenho

Ator: Professor.

Descrição: O professor observa a interação do aluno com o sistema em sala de aula ou ambiente de estudo, identificando quais tabuadas o aluno apresenta mais dificuldade.
